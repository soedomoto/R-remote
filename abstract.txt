BPS - Statistics Indonesia adalah lembaga statistik resmi di Indonesia. BPS mempunyai tugas utama untuk menyediakan statistik dasar yang digunakan pemerintah untuk menyusun kebijakan. Untuk mendukung fungsi dan kewenangannya, BPS melakukan berbagai macam pengumpulan data, baik itu data yang bersifat primer maupun sekunder, baik melalui sensus maupun survei. Ketersediaan data yang cukup lengkap di BPS, baik dari sisi variasi maupun jumlah, tidak didukung oleh ketersediaan analis yang memadai di BPS, sehingga banyak informasi yang potensial untuk diektrak dari data tersebut.

Salah satu cara untuk meningkatkan manfaat dari data tersebut adalah dengan melibatkan analis dari lembaga lain, seperti LIPI, Akademisi, maupun peneliti. Namun, dengan membuka akses data secara luas juga mempunyai efek negatif, yaitu meningkatkan potensi penyalahgunaan data. Untuk mengatasi masalah tersebut, saya mengembangkan sebuah tools yang menyediakan workspace bagi para analis diluar BPS untuk dapat melakukan analisis tanpa perlu memiliki datanya. Tools yang dibangun digunakan untuk mengeksekusi perintah R secara remote.

Tools ini menggunakan interface web notebook, seperti jupyter dan shiny, Sehingga analis tinggal mengetikkan perintah R dan hasilnya akan muncul dibawahnya. Setiap user akan disediakan sebuah workspace dengan sebuah session didalamnya, sehingga masing-masing user dapat melakukan analisisi secara independen. Tools aplikasi juga menyediakan helper yang menyediakan list data dan modul yang dapat digunakan oleh pengguna.
